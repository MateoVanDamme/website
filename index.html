Computer Vision

(Sm)Art Path: indoor localization through painting recognition

This UGent project uses computer vision algorithms to figure out the location of the user in a museum based on a video. The user walks around the museum and films his surroundings. Based on this video and a known database of paintings an algorithm caclulates the probable location. An estimation for the path is also presented. Topics include: sytem design, machine learning, computer vision

Computer Graphics

Final year project for my Computer Graphics course

This project uses the Three.Js framework to bring a virtual world to life. The project is available online and includes multiple demos. Besides the purely graphical shaders topics also include boids (swarm behavior), procedural terrain generation with LOD's and general system design. Balancing visual appearance with performance was key. Topics include Three.js, algorithms, system design and computer graphics
